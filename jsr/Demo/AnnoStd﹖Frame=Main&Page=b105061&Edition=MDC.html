5.4.33 Cancel job

(Operation type 63) cancels the preceding job request from a client program if the process has not yet been initiated.

The 1 field of the cancel job request shall be:

  1. Job request number: <LI> the job request number from the response to the job request that is being canceled.
<LI>
Job request #

Figure 58 – Cancel job request

The cancel job response shall consist of a header only. The server shall indicate a job cancel error when process initiation has completed or has irrevocably begun. That is, a successful cancel job transaction shall assure the agent that the requested process did not and will not execute.
