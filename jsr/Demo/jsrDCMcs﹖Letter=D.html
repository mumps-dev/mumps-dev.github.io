DICOM Terms and Abbreviations

DA

DA: Abbreviation for a Value Representation: Date. For details see here.

DD

DD: Abbreviation for a type of modality: Duplex Doppler.

DF

DF: Abbreviation for a type of modality: Digital Fluoroscopy.

DG

DG: Abbreviation for a type of modality: Diaphanography.

DICOMDIR

DICOMDIR: The standard directory structure specified for DICOM media exchange.

DIMSE

DIMSE: DICOM Message Service Element, the DICOM set of commands (e.g. C-STORE, C-FIND, etc.).
DIMSE-C for Composite elements, DIMSE-N for Normalized.

DM

DM: Abbreviation for a type of modality: Digital Microscopy.

DS

DS: Abbreviation for a Value Representation: Decimal String. For details see here.

DS: Abbreviation for a type of modality: Digital Subtraction Angiography.

DSA

DSA: Digital Subtraction Angiography

DT

DT: Abbreviation for a Value Representation: Date/Time. For details see here.

DX

DX: Abbreviation for a type of modality: Digital Radiography.

Data dictionary

Data dictionary: A registry of DICOM data elements which assigns a unique tag, a name, value characteristics, and semantics to each data element.

Data element

Data element: A unit of information as defined by a single entry in the data dictionary.

Data element tag

Data element tag: A unique identifier for a Data Element composed of an ordered pair of numbers (a Group Number followed by an Element Number).

Data set

Data set: Exchanged information consisting of a structured set of attribute values directly or indirectly related to information objects. The value of each attribute in a data set is expressed as a data element.

Data stream

Data stream: The result of encoding a data set using the DICOM encoding scheme (data element numbers and representations as specified by the data dictionary).

Dmax

Dmax: The maximum density used by a device to render an image.

Dmin

Dmin: The minimum density used by a device to render an image.
