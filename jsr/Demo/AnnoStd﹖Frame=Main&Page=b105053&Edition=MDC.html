5.4.25 Close all devices

(Operation type 44) The request consists of a header only, indicating that all devices owned by all clients on the client node shall be relinquished.

NOTE – This operation is included for the convenience of implementers – there is no requirement that the request be sent in any specific condition.

The response shall consist of a header only. If output to any of the devices owned by processes on the client node has not finished, the server shall return the output pending error condition and shall not relinquish that device. Whether to relinquish other devices whose output has finished is left to the implementer.
