6.4.4 BLDMODE elattribute

Definition

For a build-mode window BLDMODE specifies the actions applicable to the gadget. If the value of BLDMODE includes MOVE, the gadget can be moved within the parent window's viewport by user action. If the value of BLDMODE includes SIZE, the gadget can be resized within the parent window's viewport by user action. If the value of BLDMODE is emptystring, the gadget cannot be selected by user action and attempts to specify it as a subnode of the parent window's VALUE wattribute are ingnored.

For a window that is not in build mode, the elattribute may be set and referenced but is ignored.

Domain buildops
Access Create, Reference
Default For a build-mode window: "MOVE,SIZE"
For a window not in build mode: None
