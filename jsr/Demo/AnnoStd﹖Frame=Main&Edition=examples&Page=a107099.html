$NEXT

M[UMPS] by Example

☞

Introduced in the 1977 ANSI M[UMPS] language standard; removed in the 1995 ANSI M[UMPS] language standard.
This function is obsolete. Use the function $Order instead.

Be careful when converting code from using the function $Next to using $Order: code fragments referring to loopcounter<0 and loopcounter=–1 may get confused when such values are intended to be "exit" criteria, but also actually occur in the data being traversed.
