4.1.8 FSIZE dattribute

Definition FSIZE indicates the default value for the FSIZE wattribute for windows that are children of the display.
Domain fontsize
Access Modify, Reference
Default Platform. The font size provided by the platform when requested for a FFACE of "M,DEFAULT", a FSIZE of 12 and a FSTYLE of "NORMAL".
