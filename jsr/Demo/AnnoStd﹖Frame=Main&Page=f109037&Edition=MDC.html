9.5.16 PPOS einfoattribute

Definition PPOS identifies the pointer position. If the event is associated with a gadget, the position is measured with respect to the gadget's origin and is expressed in the unit of measure specified by the gadget's UNITS elattribute. Otherwise, the event is associated with a window and the position is measured with respect to the origin of the application area and is expressed in the unit of measure specified by the window's UNITS wattribute.
