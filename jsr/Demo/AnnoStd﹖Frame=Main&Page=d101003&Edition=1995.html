1.3 Scope of this binding

This is a binding of MUMPS using external references (cf. ANSI X11.1–1995 section 5.1.6.3). This binding specifies the interface of ANS MUMPS (using external references) to the X Window System Routine Libraries as produced by the X Consortium, the Intrinsic Toolkit so provided, and the Motif Widget and Routine set as produced by the Open Software Foundation, all of which have work underway to standardize them. The methodology cited in this binding will work for any release of the X Window System. Individual calls may not be present in all of the releases of the X Window System. A version of MUMPS syntax for the X11R4 release of the X Windows System is available in the user contributions area of the X Window System sources.
