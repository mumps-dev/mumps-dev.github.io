7.1 Draw Command Numbers

drawcommands are stored as immediate descendants of the DRAW elattribute using drawnum subscripts. The drawing that appears is that which would result from sequentially performing all drawcommands for all drawnum subscripts, beginning with the lowest valued drawnum.

If an application attempts to assign a drawnum subscript which has a negative value, an error occurs with $ECODE containing ",M?2," and $EREF containing a value that indicates the ssvn reference for which the error occurred.
