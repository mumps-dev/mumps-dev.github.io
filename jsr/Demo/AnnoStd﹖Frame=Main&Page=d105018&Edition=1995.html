5.5 XMUMPS Specific Syntax

&XMUMPS.UnregisterCallback(xpointer)

Disables disables translation (if any) for labelref associated with the xpointer used as the third argument of &XTOOLKIT.XtAddCallback and related routines. This routine frees any memory allocated by the associated &XMUMPS.RegisterCallback call for use by the system.
