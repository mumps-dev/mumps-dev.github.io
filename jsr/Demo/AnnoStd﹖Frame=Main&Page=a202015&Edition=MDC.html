2.8 Character strings

Character string length is limited to 32,767 characters for local variables, 510 characters for global variables, and 32,767 characters for structured system variables. The characters permitted within character strings must include those defined in the ASCII Standard (ANSI X3.4–1986). If a string's length exceeds an implementor's limit, an error condition occurs with ecode = "M75".
