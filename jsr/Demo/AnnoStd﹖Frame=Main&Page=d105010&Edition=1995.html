5.5 XMUMPS Specific Syntax

&XMUMPS.SetValue(xmember,expr)

Modifies the value of the element of the xtype referenced by xpointer where xmember is a reference to the xtype. It is erroneous to use an xmember for which no xname exists. Any type conversions required are implementation details. Data is assumed to be numeric.
