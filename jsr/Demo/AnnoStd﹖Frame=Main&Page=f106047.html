6.4.27 LINKACTION elattribute

Definition

Assigning a value to LINKACTION causes an action to occur in the element's Data Exchange link, if any. The assignment has no effect if the element does not have a Data Exchange link established. Actions occur asynchronously, and cause a LINKACTION event upon completion or timeout.

POKE The application at the other end of this element's Data Exchange link is asked to assign the text in the element's VALUE elattribute to the current item
REQUEST The application at the other end of this element's Data Exchange link is asked to send the text of the current item. The text received is assigned to the element's VALUE elattribute.
EXECUTE The application at the other end of this element's Data Exchange link is asked to execute the command in the element's VALUE elattribute.
emptystring No action is required.

Reference to this elattribute always returns emptystring.

Domain deaction
Access Create, Modify, Reference
Default Not applicable
