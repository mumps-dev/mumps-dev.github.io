5.4.30 SCROLL wattribute

Definition If SCROLL is TRUE and a portion of the application area lies outside the window viewport, a horizontal and/or a vertical scroll bar is present, enabling the user to scroll the application area. Otherwise, scroll bars for scrolling the application area are not present, if the underlying windowing platform permits them to be omitted.
Domain truefalse
Access Create, Reference
Default TRUE
