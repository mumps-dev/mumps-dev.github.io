7.1.6.5.50 $%SEC^MATH

SEC^MATH : REAL ( X : REAL , PREC : INTEGER : O )

$%SEC^MATH returns the value of the trigonometric secant of X. The value of X is expressed in radians. $%SEC^MATH(X) ≤ –1 or $%SEC^MATH(X) ≥ 1. The number of significant digits in the secant is specified by the optional parameter PREC. If not specified, a default value of 11 digits is assumed for PREC.
