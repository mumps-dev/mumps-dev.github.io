5.5 XMUMPS Specific Syntax

The routines in this package are specific to MUMPS. These routines do not generally map to calls provided on X Window Systems, but rather are calls needed for most MUMPS programmers to readily access the X Window System without requireing that the external data structures be integrated into the MUMPS implementation.
