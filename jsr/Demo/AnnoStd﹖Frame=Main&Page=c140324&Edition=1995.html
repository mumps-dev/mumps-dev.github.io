A null parameter leaves the ASF unchanged. The default ASF for all parameters is INDIVIDUAL. Please note the ANSI standard for the default ASF is INDIVIDUAL while the ISO default is BUNDLED. A GKS application program should be very cautious in leaving the ASF to the default settings.
