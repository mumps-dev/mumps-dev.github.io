7.1.6.5.17 $%CLOG^MATH

CLOG^MATH : COMPLEX ( Z : COMPLEX , PREC : INTEGER : O )

$%CLOG^MATH returns the logarithm of the complex number Z; Re ($%CLOG^MATH(Z)) can be any number, - ≤ Im ($%CLOG^MATH(Z)) ≤ π. The number of significant digits in the complex logarithm is specified by the optional parameter PREC. If not specified, a default value of 11 digits is assumed for PREC. If Im (Z) = 0, then Re (Z) > 0.
