9.1.11 Applicability of Event Types

The applicability of etypes to window, element, and choice types is specified by Table 4. A "Yes" entry indicates that the occurrence of the event type with the window, element, or choice type is defined. An empty entry indicates that the occurrence of the event type with the window, element, or choice type is reserved. implementationattribute values are applicable to all window, element and choice types.
