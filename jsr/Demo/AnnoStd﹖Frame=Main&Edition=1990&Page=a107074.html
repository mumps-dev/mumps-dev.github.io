2.2.6 Intrinsic Special Variable Name svn

Syntax Definition
$T[EST] $T contains the truth value computed from the execution of an If command containing an argument, or an Open, Lock, Job, or Read command with a timeout (see 2.2.9, 2.2.10, and 2.6.3).
