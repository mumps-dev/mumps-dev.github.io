2.2.6 Intrinsic Special Variable Name svn

Syntax Definition
$X $X has a nonnegative integer value which approximates the value of a carriage or horizontal cursor position on the current line as if the current I/O device were an ASCII terminal. It is initialized to zero by input or output of control functions corresponding to CR or FF; input or output of each graphic adds 1 to $X (see 2.5.5 and 2.6.18).
