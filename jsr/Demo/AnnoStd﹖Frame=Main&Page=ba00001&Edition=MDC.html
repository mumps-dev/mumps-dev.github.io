Annex A
(Normative)
Conformance

This annex defines levels of conformance to this standard and prescribes information that implementers shall provide to document the conformance of an implementation.

NOTE – The term program used in this annex does not refer to a M[UMPS] program or routine. It refers to the program with which a M[UMPS] or other product implements the protocol defined in this standard.
