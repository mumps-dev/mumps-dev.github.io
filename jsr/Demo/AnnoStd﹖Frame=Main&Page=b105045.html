5.4.17 Extquery

(Operation type 74) The extquery operation fetches extended information about the global reference following the global reference <LS> of its request. ANSI/MDC X11.1, 7.1.5.15 defines the ordering sequence.

<LS>
<LI> Global Reference

Figure 37 – Extquery request

The extquery response shall include the reference and the value of the following global reference. The fields of the extquery response and their sequence shall be:

  1. Global reference: <LS> the following global reference. If there is no following global reference, this field shall equal the empty string.
  2. Global value: <LS> the value of the global variable defined by global reference. If global reference equals the empty string, this field shall equal the empty string.
