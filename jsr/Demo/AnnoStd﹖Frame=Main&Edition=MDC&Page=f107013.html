7.2.10 FSIZE Draw Command

    drawfsize::= expr V FSIZE , fontsize

The drawfsize drawcommand specifies the current font size. The specified font size is used by subsequent drawcommands until another drawfsize drawcommand is specified. Prior to the first drawfsize drawcommand for a drawing, the current font size is specified by the gadget's FSIZE elattribute. Requesting of a new font from the underlying windowing platform (if any) is deferred until the first subsequent drawcommand requiring a font.
