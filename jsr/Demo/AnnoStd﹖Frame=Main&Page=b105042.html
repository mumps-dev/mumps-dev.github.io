5.4.14 Extget

(Operation type 70) The extget operation fetches extended information about a global reference given in its global reference field <LS>.

<LS>
<LI> Global Reference

Figure 31 – Extget request

The extget response shall include the following subscript on the same level as the final subscript of the requested global reference, or the following global name; the $data value of the requested global reference; and its value, if any.

The fields of the extget response and their sequence shall be:

  1. Order value: <SS> the following subscript or the following global name as specified in 5.4.10.
  2. $data value: <SI> the $data value of the requested global reference. ANSI/MDC X11.1, 7.1.5.3, defines the $data values that indicate the presence of a global value and descendants.
  3. Global value: <LS> the value of the requested global variable. If the $data value indicates that no value exists, this field shall equal the empty string.
